Hawks OS

An approval-first agentic job search operating system.

Hawks scouts trusted roles, validates sources, scores fit, dedupes noise, builds context packs, and keeps a decision ledger — so job search becomes inspectable workflow, not prompt-only browsing.

GitHub repo Architecture Trust model

Scout as Code

Fit scoring

Every shortlist includes evidence, target signals, risks, and a score humans can inspect.

Approval gate

Hawks prepares and explains. It does not blindly submit applications.

Run ledger

Agent runs should leave a durable trace: inputs, decisions, outputs, and next action.
